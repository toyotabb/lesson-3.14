## git add

---

Команда **git add *[файл]*** Переносит изменения из рабочего каталога в раздел проиндексированных файлов. В этом разделе можно подготовить снимок состояния, прежде чем сделать коммит изменений в официальную историю.

Чтобы добавить все файлы в каталоге в индекс (кроме игнорируемых), используйте команду:

```bash=
git add .
```
